aoc.interval.intervals_diff(ints1: Set[Tuple[float, float]], ints2: Set[Tuple[float, float]]) → Set[Tuple[float, float]][source]#

For two sets of disjoint open-closed intervals on the real line ints1 and ints2, compute the set difference ints1 - ints2.

Parameters:

  • ints1 (set[Interval]) – First set of intervals.

  • ints2 (set[Interval]) – Second set of intervals.

Returns:

ints1 - ints2.

Return type:

set[Interval]
